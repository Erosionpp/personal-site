# 项目经历

## 数据/金融分析项目

### Education & Care Services Analysis
分析澳大利亚 17,000+ 幼托服务的数据，识别服务缺口和质量趋势。使用 Python 进行数据清洗和地理空间可视化。发现服务集中在 NSW 和 VIC，大部分仅达标而非超标，独立幼儿园表现最佳。

### VIX Volatility Forecasting with Basis Function Models
比较四种基函数回归模型预测市场波动率（VIX），使用 8,920 个观测值（1990-2025）。高斯 RBF 模型表现最优，对数变换将 MSE 降低约 14%。预测 2026-2028 VIX 回归长期均值 18-19。

### Thailand Digital Nomad Visa Policy Analysis
数据驱动的泰国数字游民签证政策分析。构建了包含停留分布、季节性趋势、城市分布和区域比较的 Tableau 仪表板。发现大多数游民停留 30-90 天，泰国中位停留时间落后越南和马来西亚。

## 设计项目

### VAPORFORMER — 变形玩具设计（毕业设计）
以蒸汽波美学为灵感的变形玩具，将复古未来主义艺术与机械工程融合。提取了棱镜几何、怀旧引用和天线特征三个核心蒸汽波设计元素。使用 Rhino 建模，ABS + PLA 双材料体系。可在机器人与飞行器形态间自由切换，推出多款限定配色。

### INNO — 无接触消毒设备
为地铁站设计的非接触式喷雾消毒装置。经过用户研究确定最佳喷雾高度（100cm），适合成人、儿童和轮椅使用者。T 型立式设计，具备自动感应、LED 反馈环和可更换瓶系统。

### REVO — 智能消防头盔
集成 AR 显示、热成像和多传感器融合的消防头盔。梳理消防员任务流程，识别四大痛点。针对建筑火灾、工业火灾和野火设计了场景化 UI 适配，配备自适应可见度模式。
